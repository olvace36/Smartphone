Instruction to create your own theme:
    - Copy folder 'default' to a new one and rename it to whatever you want 'my_theme'
    - Copy your images into the new folder and rename it to the respective texture (app_camera, phone_empty,...)
    - Restart the game. You will see 'my_theme' option appear in the Setting -> Themes

Requirement:
    - Phone, phone_empty and phone_background.png must have the exact pixel size as default
    - Apps icon can be any size (1024*1024). However they should be square and larger than 84*84 for best appear
    - File name must exactly match with default
